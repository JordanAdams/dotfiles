## Requirements

- [Homebrew](https://brew.sh/)


## Setup

1. Clone this repo
2. Run `./link` to link configuration files
3. Run `./install` to perform setup

## Support

- Homebrew
- ZSH
- Neovim
- Languages (JS, Rust, Python, etc.)
- Alacritty
- Tmux
